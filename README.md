# Liri-node-app
LIRI is a Language Interpretation and Recognition Interface

It can take in one of the following commands:

    * `spotify-this-song`

    * `movie-this`

    * `do-what-it-says`


## Technologies Used

* Node.js
* Npm
* Spotify API
* Omdb
* Require
* fs
* Figlet
* Columnify
* Keys
* dotenv package
